# 皮肤管理 教学法笔记

基于 `../../../references/pedagogy-framework.md`。针对「护肤」这个**生理 + 行为 + 强信息噪声**技能的落地。

## 最该练的是『分辨』，不是『记成分』
护肤水深、营销噪声极大。难的不是背成分，是**分辨「有证据的基础 vs 智商税 vs 该看医生」**。
所以刻意练习重点放在「让你写出方案 / 选择 → 我核对逻辑与证据」，以及「把营销话术翻译成成分事实」。

## 卡点地图（route 5）
- **想堆产品 / 叠满步骤** — `build-routine` + `consistency-habits` 反复强调 less is more。
- **一次上太多活性物刷坏屏障** — `patch-test-intro` + `over-exfoliation-barrier` 用慢引入 + 修复急救纠正。
- **过度清洁** — `cleansing` 用「洗后紧绷」信号点出。
- **防晒用量不足** — `sun-protection` 用两指法量化。
- **把皮肤病当护肤问题硬抠** — `target-concerns` + `when-see-derm` 划清边界。

## 间隔重复清单（进 spaced_queue）
防晒两指法用量 · 活性物早晚搭配禁忌 · 屏障受损信号 · ABCDE 痣自查 · 「基础三件 > 精华」。

## 节奏 + 安全（route 8 + 诚实边界）
护肤是慢功夫，皮肤适应要时间，频繁换产品反而坏事。每次只小改一处。
**AI 看不到你的皮肤；皮肤病、可疑痣的第一动作是面诊皮肤科，不是继续上课。**
