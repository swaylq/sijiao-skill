# 调研笔记（8 路汇总）· skincare（科学护肤）

> 参考样本：从皮肤科医生 / 科普正典蒸出以演示管线。完整 per-route 拆分见 `rust-learn`；此处汇总。**本样本非医疗建议。**

- **1 知识地图 / 先修**：皮肤/屏障 → 判断肤质诉求 → 三大件（清洁/保湿/防晒）→ 搭流程 → 活性物 + 斑贴慢引入 → 屏障修复 / 针对诉求 → 何时看医生。来源：皮肤科科普的层级结构 `[primary]`。
- **2 正典**：Lab Muffin Beauty Science（Michelle Wong，化学博士）· American Academy of Dermatology (AAD) · Dr. Dray（皮肤科医生 YouTube）· Paula's Choice 成分词典 · INCIDecoder。三处以上共识。
- **3 高手路径**：共识「less is more」—— 基础三件做扎实 + 慢引入活性物，远胜追新品堆步骤。
- **4 刻意练习**：核心是「写出你的方案 / 选择 → 我核对逻辑与证据」+ 把营销话术翻译成成分事实。环境：成分查询站（INCIDecoder）+ 产品成分表。
- **5 卡点**：想堆产品、一次上太多活性物刷坏屏障、过度清洁、防晒用量不足、把皮肤病当护肤硬抠。
- **6 评估**：novice = 懂屏障 + 会判肤质 + 做好三件；adv beginner = 会搭极简流程 + 认活性物 + 慢引入；competent = 能按诉求选成分、识别 + 修复屏障受损、知道何时就医。
- **7 反馈 / 安全**：皮肤病 / 可疑痣 → 皮肤科面诊；过敏 / 刺激 → 停用就医。AI 看不到你的皮肤，只能批方案逻辑。
- **8 动机**：慢功夫，皮肤适应要时间；频繁换产品坏事；每次只小改一处 + 极简才坚持得住。

**安全声明**：本样本是科普教练，不构成医疗诊断或治疗建议；皮肤病、可疑皮损请面诊皮肤科。
