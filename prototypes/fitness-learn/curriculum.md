# fitness — 学习路径

## 阶段：novice

### 训练底层：渐进超负荷、训练量、恢复  `(training-principles)`
- 目标（understand）：解释肌肉/力量增长的机制：渐进超负荷 + 足够训练量 + 恢复，三者缺一不可
- 目标（analyze）：破除『不练到力竭/不酸痛就没用』『某动作能局部减脂』等迷思
- 练习（drill）：用自己的话写出『渐进超负荷』是什么、为什么不加量就不长，发我核对有没有把『练得累』当成『有效』

### 设定目标与基线、现实预期  `(set-goals-baseline)`
- 先修：training-principles
- 目标（evaluate）：明确目标（增肌 / 力量 / 体能 / 健康），不要什么都想要
- 目标（understand）：建立现实预期（自然增肌速率有限，没有捷径）+ 记录起始基线
- 练习（drill）：写下你的主目标 + 起始基线（体重、几个关键动作能做多少），发我；我帮你看目标是否现实、是否贪多
- ⚠️ 诚实边界：开始规律训练前，有心血管病 / 高血压 / 关节伤病 / 孕期等情况，先让医生评估能不能练、能练什么。本课不替代医疗评估。

### 基础动作模式：推 / 拉 / 蹲 / 铰链 / 核心  `(movement-basics)`
- 先修：training-principles
- 目标（understand）：认识五大基本动作模式，理解『先学会模式，再加重量』
- 目标（apply）：用自重 / 轻负荷把每个模式做顺
- 练习（drill）：每个动作模式各选一个自重动作做几组，记录感受（哪里发力、哪里别扭），发我；我帮你判断是模式问题还是力量不足

### 动作标准与安全：形先于重量  `(form-safety)`
- 先修：movement-basics
- 目标（understand）：理解为什么『标准动作 + 小重量』优于『大重量 + 歪扭』
- 目标（apply）：建立热身 + 渐进加重的习惯，认得明显的危险信号
- 练习（drill）：录一段你某个动作的侧面视频自己对照教学要点找 3 个问题，把你发现的问题发我讨论（我只能基于你的描述给思路，看不到视频本身）
- ⚠️ 诚实边界：**动作标准度我隔着屏幕看不准** —— 我能讲要点、能基于你的描述给思路，但真要纠正动作，请对照教学视频自录侧面录像、或找线下教练 / 私教，尤其大重量动作。

## 阶段：advanced_beginner

### 恢复：睡眠、训练间隔、过度训练  `(recovery-sleep)`
- 先修：training-principles
- 目标（understand）：理解『肌肉在恢复时长，不是在训练时长』，安排足够休息
- 目标（analyze）：识别恢复不足 / 过度训练的信号（持续疲劳、力量倒退、睡眠差）
- 练习（drill）：记录一周睡眠 + 训练日 + 主观疲劳（1-10），发我；我帮你看是不是恢复不够还硬练

### 训练营养：蛋白、热量、增肌减脂取舍  `(nutrition-for-training)`
- 先修：set-goals-baseline
- 目标（apply）：算出蛋白目标（约 1.6-2.2 g/kg）和增肌/维持/减脂的热量方向
- 目标（understand）：理解『增肌要略盈余、减脂要赤字，很难同时大幅兼得』
- 练习（drill）：根据你的目标算出蛋白目标 + 热量方向，列一天大致吃法，发我核对

### 选一套新手训练计划  `(beginner-program)`
- 先修：set-goals-baseline, movement-basics
- 目标（understand）：理解新手为什么适合『全身 2-3 次/周』而非花哨分化
- 目标（apply）：选定一套有信誉的新手计划并排进日程
- 练习（project）：选一套新手计划，把它排进你下周的具体日程（哪天练、练什么），发我；我帮你看频率/恢复是否现实、是否一上来就贪多

### 渐进超负荷与训练记录  `(progressive-overload)`
- 先修：beginner-program
- 目标（apply）：执行线性进步（每次/每周小幅加重量或次数）
- 目标（apply）：用训练日志追踪，靠数据而非感觉判断有没有进步
- 练习（drill）：记录你这周每个动作的重量×组×次，写出下次每个动作怎么加，发我核对加得是否过激

### 复合动作详解：深蹲 / 硬拉 / 卧推 / 划船 / 推举  `(compound-lifts)`
- 先修：form-safety, beginner-program
- 目标（analyze）：说清每个复合动作练哪、关键技术要点、常见错误
- 目标（apply）：为复合动作安排合理的热身组与起始重量
- 练习（drill）：挑一个复合动作，写出它的关键要点 + 你自查的常见错误清单，发我讨论
- ⚠️ 诚实边界：大重量复合动作（尤其硬拉、深蹲）**形不对容易伤腰/膝** —— 起步阶段强烈建议找教练或对照录像，别凭感觉冲重量。

## 阶段：competent

### 防伤与疼痛处理  `(injury-prevention)`
- 先修：form-safety, recovery-sleep
- 目标（analyze）：区分正常的延迟性酸痛（DOMS）与受伤信号
- 目标（evaluate）：知道什么时候该减量 / 停训 / 就医，而不是『忍痛练』
- 练习（drill）：列出你能想到的『正常酸痛 vs 该警惕』各 3 条，发我核对；记住：尖锐/关节/持续疼或麻木，停训找医生
- ⚠️ 诚实边界：**关节痛、尖锐痛、持续痛、麻木刺痛不是『正常酸痛』** —— 出现就停下该动作，必要时看医生 / 理疗师，别用本课自我诊断硬扛。

### 有氧 / 体能整合  `(cardio-conditioning)`
- 先修：beginner-program
- 目标（apply）：在不显著拖累增肌的前提下安排有氧 / 体能
- 目标（evaluate）：按目标决定有氧的量与时机
- 练习（drill）：根据你的目标，写一个把有氧塞进当前计划的方案（频率/时机），发我看会不会冲突恢复

### 长期坚持：把训练变成习惯  `(consistency-longterm)`
- 先修：beginner-program
- 目标（create）：用现实可持续的频率 + 习惯设计避免三分钟热度
- 目标（evaluate）：破除『追完美计划却不练』，理解坚持 > 最优
- 练习（drill）：写一条让你不翘训练的 if-then + 环境设计（如把训练装备/健身房路线固定），发我
- 里程碑：独立设计/选择并执行一套适合自己目标的训练计划，渐进超负荷 4-8 周并在卡住时正确调整 — 会了的标志：能独立选/排计划、渐进加量、动作安全、平台期会调、并长期坚持，且能讲清每步原理

### 平台期与计划调整  `(program-adjustment)`
- 先修：progressive-overload, recovery-sleep
- 目标（analyze）：区分真平台（连续几周加不动）与正常波动
- 目标（evaluate）：用 deload / 检查恢复营养 / 换计划 三步定位并破局
- 练习（drill）：假设你某动作 3 周加不动了，按『先查恢复/营养 → 再考虑 deload → 必要时换计划』写出你的排查步骤，发我核对顺序
