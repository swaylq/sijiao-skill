# 调研笔记（8 路汇总）· fitness（力量 / 增肌训练）

> 参考样本：从证据导向的训练科学正典蒸出以演示管线。完整 per-route 拆分见 `rust-learn`；此处汇总。**非医疗 / 训练处方。**

- **1 知识地图 / 先修**：训练原理（渐进超负荷 / 量 / 恢复）→ 目标基线 + 动作模式 + 形与安全 → 选计划 / 加量 / 复合动作 / 营养 / 恢复 → 平台调整 / 有氧 / 防伤 / 坚持。来源：训练科学层级 `[primary]`。
- **2 正典**：r/Fitness Wiki（thefitness.wiki）· Stronger By Science（Greg Nuckols）· Renaissance Periodization（Mike Israetel）· Jeff Nippard · Barbell Medicine（医生 + 举重者）。三处以上共识。
- **3 高手路径**：共识「新手别抄花哨分化，全身 2-3 次/周 + 渐进超负荷 + 记录」最有效；形先于重量。
- **4 刻意练习**：核心是「写出你的计划 / 加量方案 → 我核对逻辑」+ 训练日志。动作部分转『录像对照 + 教练』（AI 看不到）。环境：健身房或家用器械 + 训练记录 app。
- **5 卡点**：冲重量不顾形、抄花哨分化、恢复不够硬练、忍痛练、追完美却不练。
- **6 评估**：novice = 懂原理 + 会基本动作模式 + 知道形先于重量；adv beginner = 会选/排计划 + 渐进加量 + 认复合动作；competent = 能破平台、整合有氧、判断伤病、长期坚持。
- **7 反馈 / 安全**：动作 → 录像对照 / 线下教练；疼痛 / 伤病 → 医生 / 理疗师；疾病 → 先医生评估。AI 看不到你的动作和身体，只能批计划逻辑。
- **8 动机**：慢功夫，渐进才安全；坚持 > 最优计划；习惯 / 环境设计防三分钟热度。

**安全声明**：本样本是科普教练，不构成医疗或个性化训练处方；动作纠正需线下，伤病 / 疾病请就医。
