# fitness-learn — 健身私教（参考样本）

sijiao-skill 样本之六，第三个非纯认知技能（生理 + 行为）—— 也是「**诚实降级**」最极端的案例：
健身教练的核心是**盯动作**，而 **AI 隔屏根本看不到你的动作**。所以这个样本把『动作标准』整块诚实标注
转录像对照 / 线下教练，AI 只负责能核对的部分（计划 / 加量 / 营养 / 恢复 / 判断）。

| 文件 | 说明 |
|------|------|
| `curriculum.json` → `curriculum.md` | 13 模块 novice→competent（训练原理 → 动作/形 → 计划/渐进/复合动作 → 平台/防伤/坚持）；**4 个模块带 `honest_limit`**（医学评估 / 看不到动作 / 大重量伤病 / 酸痛 vs 受伤） |
| `SKILL.md` | 私教：证据导向，**「我看不到你的动作」写在诚实边界最显眼处** |
| `learner-state.example.json` | 档案示例（学习者卡在「复合动作」+ 想冲重量 / 硬拉弓背） |
| `references/pedagogy.md` · `references/research/notes.md` · `meta.json` | 教学法 / 8 路汇总 / 元数据 |

> ⚠️ **非医疗 / 训练处方。** 参考样本，课程从训练科学正典（r/Fitness wiki · Stronger By Science · RP · Jeff Nippard · Barbell Medicine）蒸出。动作纠正需线下、伤病请就医。`update 私教 fitness` 做完整刷新。
