# Route 2 — 正典教材与课程 (rust)

≥3 处独立推荐、几乎每份「如何学 Rust」清单都点名的：

- **The Rust Programming Language（TRPL，「the book」）** `T02-S001` https://doc.rust-lang.org/book/ — 官方、免费、本课主线 [primary]
- **Rustlings** `T02-S002` https://github.com/rust-lang/rustlings — 官方交互式 drill，配套 the book [primary]
- **Rust by Example** `T02-S003` https://doc.rust-lang.org/rust-by-example/ — 例子驱动速查 [primary]
- **std 文档** `T02-S004` https://doc.rust-lang.org/std/ [primary]

精通区（超出本课天花板，诚实边界里指路）：《Programming Rust》(O'Reilly)、《Rust for Rustaceans》(Jon Gjengset)、Comprehensive Rust (Google)。
