# Route 3 — 高手当年怎么学的 (rust)

真实路径共识（来自社区大量「我是怎么学 Rust 的」复盘）：

- **主线**：the book + Rustlings **并行**（读一章 → 做对应 rustlings），别只读不练。
- **关键捷径**：生命周期标注**别上来硬背**——等编译器逼你的时候再学，理解快得多。
- **「早该做」**：尽早建一个真实小项目（CLI），逼自己把零散知识串起来。
- **「早该跳过」**：第一周别为了「不写 clone」死磕——先 clone 跑通，胜任后再优化。

来源：`T03-S001` 社区学习复盘合集（forum / 博客）· `T03-S002` Jon Gjengset《Crust of Rust》系列（进阶视角）。
