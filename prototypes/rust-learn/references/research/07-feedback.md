# Route 7 — 反馈与社区 (rust)

AI 能批编译/逻辑/风格；「品味」层面的 code review 去这些地方求真人：

- **users.rust-lang.org** `T07-S001` https://users.rust-lang.org/ — 官方论坛，对初学者友好，有专门的 code review 版块。
- **r/rust** `T07-S002` https://www.reddit.com/r/rust/ — 周常「easy questions」帖。
- **Rust Discord / Zulip** `T07-S003` — 实时答疑。

honest_limit 落点：本课所有模块 AI 都能批改（认知甜区），故各模块 `honest_limit=null`；唯「工程品味/架构」层面引导到真人 review。
