# 调研笔记（8 路汇总）· english-reading

> 参考样本：从英语阅读教学法蒸出以演示管线。完整 per-route 拆分见 `rust-learn` 样本；此处汇总。

- **1 知识地图 / 先修**：高频词 + 句法骨架 → 简单句 → 长难句 → 段落/逻辑 → 推断/语气/论证 → 体裁策略。两条早期并行线（词汇、句法）汇入阅读机制。
- **2 正典 / 资源**：NGSL 高频词表 `[primary]` · Oxford Bookworms 分级读物 · Cambridge English 阅读 · Breaking News English · 真实语料 (Economist/Aeon/Simple Wikipedia)。
- **3 高手路径**：共识「海量可理解输入（extensive reading）+ 高频词检索」远胜啃语法。
- **4 刻意练习**：AI 现场生成分级文章 + 出题 + 证据核对；分级读物逐级升到真实语料。环境：纯文本，无需工具。
- **5 卡点**：逐字翻译、长句找不到主干、生词就停、推断凭感觉。
- **6 评估**：novice = 不查词读懂 Stage 1；adv beginner = 拆长句 + 抓主题句；competent = 读懂未简化真实长文的主旨/推断/语气，正确率 ≥80%。
- **7 反馈**：AI 能批阅读理解（含证据核对，强甜区）；口语/写作产出需别的资源或真人。
- **8 动机**：4h/周到胜任约 14-20 周；词汇瓶颈靠每日小批检索 + 海量分级阅读维持。
- **理论支撑**：Krashen 可理解输入假说（i+1）。
