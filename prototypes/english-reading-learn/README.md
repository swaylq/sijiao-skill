# english-reading-learn — 英语阅读理解私教（参考样本）

learn-skill 样本之三：语言·认知甜区。验证「AI 现场生成分级材料 + 批改」这条最强闭环（口语要实时音频反馈、是 honest-limits 案例，所以选阅读不选口语）。

- `curriculum.json` → `curriculum.md` — 12 模块 novice→competent（高频词 → 长难句 → 推断/语气 → 体裁策略）
- `SKILL.md` — 私教：AI 按 `mastery` 现场生成分级文章、出题、**核对原文证据**
- `learner-state.example.json` — 档案示例（学习者卡在「长难句」+ 词汇瓶颈）
- `references/pedagogy.md`（可理解输入 i+1）· `references/research/notes.md`（8 路汇总）· `meta.json`

> 参考样本，课程从英语阅读教学法正典（NGSL / 分级读物 / Krashen / Cambridge）蒸出。
