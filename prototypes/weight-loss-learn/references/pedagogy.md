# 减肥 教学法笔记

基于 `../../../references/pedagogy-framework.md`。针对「减脂」这个**生理 + 行为混合**技能的落地。

## 最该刻意练习的不是知识，是依从
减脂知识其实简单（赤字 + 蛋白 + 力量 + 睡眠）。难的是**长期执行**。所以刻意练习重点放在
「写出你自己的方案 → 我核对数字和逻辑」和「habit / if-then 设计」，不是背理论。

## 卡点地图（route 5）
- **低估摄入** — `tracking-basics` 头号坑；用「记录 3 天我帮你反推」暴露它。
- **全或无思维** — `adherence-psychology` 专门破：破戒一餐 ≠ 整天废。
- **被营销误导**（碳水/某食物致胖、局部瘦、补剂魔法）— `nutrition-foundations` 纠正。
- **把有氧当主角、忽视力量** — `resistance-training` 强调保肌肉 = 保代谢与体型。
- **平台期慌了乱改** — `plateau-adaptation` 给固定排查顺序（先查依从 → 再核算 → 必要时 diet break）。

## 间隔重复清单（进 spaced_queue）
合理速率（0.5-1%/周）· 蛋白目标 1.6-2.2 g/kg · 趋势体重 ≠ 单点 · 平台期排查三步 · 维持红线。

## 节奏 + 安全（route 8 + 诚实边界）
慢就是快：可持续速率才不反弹。每次只小改一处习惯，别推翻全部生活。
**任何医学情况 / 进食障碍信号，第一动作是转专业人士，不是继续上课。**
