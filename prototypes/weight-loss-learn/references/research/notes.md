# 调研笔记（8 路汇总）· weight-loss（科学可持续减脂）

> 参考样本：从证据导向的营养 / 运动科学正典蒸出以演示管线。完整 per-route 拆分见 `rust-learn`；此处汇总。**本样本非医疗建议。**

- **1 知识地图 / 先修**：能量平衡是根 → 目标/速率 + 追踪 + 蛋白 → 营养/搭餐/NEAT/力量/睡眠 → 习惯/平台/依从 → 维持。来源：营养金字塔的层级结构 `[primary]`。
- **2 正典**：Examine.com（证据库）· Eric Helms《Muscle & Strength Nutrition Pyramid》· Renaissance Periodization · Precision Nutrition · James Clear《Atomic Habits》· National Weight Control Registry（维持）。三处以上共识。
- **3 高手路径**：成功长期维持者共性（NWCR）—— 规律自我监测、高蛋白、力量 + 步数、把它做成习惯而非短期节食。
- **4 刻意练习**：核心是「写出你的方案 → 我核对数字逻辑」+ 记录数据让 AI 反推。环境：一个食物记录 app + 体重秤 + 趋势工具。
- **5 卡点**：低估摄入、全或无思维、被营销误导（碳水/补剂/局部瘦）、重有氧轻力量、平台期乱改。
- **6 评估**：novice = 会算 TDEE/赤字/蛋白；adv beginner = 能搭达标的一天 + 排训练；competent = 能独立设计、破平台、过渡维持并讲清原理。
- **7 反馈 / 安全**：医学情况 → 医生 / 注册营养师；动作 → 教练 / 视频；进食障碍信号 → 精神科 / 临床营养师。AI 只批方案逻辑，批不了你的身体和实际执行。
- **8 动机**：慢就是快（0.5-1%/周才不反弹）；靠习惯 / 环境设计而非意志力；每次只小改一处。

**安全声明**：本样本是科普教练，不构成医疗建议；有医学情况或进食障碍史请先找专业人士。
