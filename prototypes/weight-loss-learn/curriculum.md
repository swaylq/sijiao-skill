# weight-loss — 学习路径

## 阶段：novice

### 能量平衡：减脂唯一的底层机制  `(energy-balance)`
- 目标（understand）：用自己的话解释「热量赤字」是减脂的必要条件，不存在魔法食物
- 目标（apply）：估算自己的维持热量（TDEE）
- 练习（drill）：用任一 TDEE 公式算出你的维持热量，再设一个每天 300-500 kcal 的赤字目标，把数字发我，我帮你核对是否在合理区间

### 设定现实目标与速率  `(set-realistic-goals)`
- 先修：energy-balance
- 目标（evaluate）：把目标定成可持续速率（约每周 0.5-1% 体重），而非「一个月瘦 20 斤」
- 目标（apply）：建立正确的基线测量（体重趋势 + 围度 + 照片，而非单日体重）
- 练习（drill）：写下你的起始体重、目标体重、和按每周 0.5-1% 算出的现实时间线；我帮你检查速率是否过激
- ⚠️ 诚实边界：有甲状腺 / PCOS / 糖尿病 / 孕期 / 在服药等情况，先让医生或注册营养师定方案 —— 本课是科普教练，不替代医疗。

### 追踪：摄入估算与体重趋势  `(tracking-basics)`
- 先修：energy-balance
- 目标（apply）：用食物记录估算每日摄入（不必精确到克，但要诚实）
- 目标（understand）：看「体重趋势」而非单点波动（水分/盐/经期会±1-2kg）
- 练习（project）：连续记录 3 天饮食 + 每天晨称体重，把数据发我；我帮你算平均摄入、和体重趋势线，指出明显低估的地方
- ⚠️ 诚实边界：AI 看不到你实际吃了什么 —— 全靠你诚实记录。估算误差（尤其油、酱、零食）是减脂停滞最常见的原因，我只能基于你报的数据反推。

### 蛋白质与饱腹感  `(protein-satiety)`
- 先修：energy-balance
- 目标（understand）：解释减脂期高蛋白为什么关键（保肌肉 + 强饱腹 + 高食物热效应）
- 目标（apply）：算出自己的蛋白目标（约 1.6-2.2 g/kg 体重）并落到每餐
- 练习（drill）：算出你的每日蛋白目标，列 5 个你爱吃的高蛋白食物及其蛋白含量，凑出一天达标的吃法发我核对

## 阶段：advanced_beginner

### 日常活动量（NEAT）与步数  `(movement-neat)`
- 先修：energy-balance
- 目标（understand）：理解 NEAT（非运动消耗）往往比「去健身房那 1 小时」对总消耗影响更大
- 目标（apply）：设一个可持续的步数 / 活动目标并融入日常
- 练习（drill）：测你近 3 天平均步数，设一个 +1500~2000 步的现实目标 + 3 个具体落地方式（如午饭后走），发我

### 睡眠与压力对食欲的影响  `(sleep-stress)`
- 先修：energy-balance
- 目标（understand）：解释睡不够 / 高压如何升高食欲、降低依从性
- 目标（apply）：落地 1-2 个改善睡眠 / 减压的具体习惯
- 练习（drill）：记录 3 天睡眠时长 + 当天饥饿程度（1-10），发我；我帮你看有没有相关，并定一个睡眠习惯

### 力量训练：减脂期最该做的运动  `(resistance-training)`
- 先修：set-realistic-goals
- 目标（understand）：解释减脂期力量训练为什么比纯有氧更重要（保肌肉 = 保代谢 + 体型）
- 目标（apply）：选一套适合新手的全身计划，每周 2-3 次
- 练习（drill）：选一套新手全身计划，排进你下周日程（具体哪天哪时），发我；我帮你看频率 / 恢复是否现实
- ⚠️ 诚实边界：动作标准度我隔着屏幕看不准 —— 跟官方教学视频或找教练纠正起步动作。有伤病 / 心血管问题先问医生或理疗师，别硬上。

### 营养基础：宏量、食物质量、不妖魔化  `(nutrition-foundations)`
- 先修：protein-satiety
- 目标（analyze）：区分宏量营养（蛋白/碳水/脂肪）各自作用，破除「碳水/某食物让你胖」的迷思
- 目标（apply）：用「多数时候吃高营养密度、灵活留 10-20% 给爱吃的」原则规划
- 练习（drill）：把你常吃的 10 样食物按「高营养密度 / 可灵活摄入」分两类，我帮你看分类是否被营销误导

### 怎么搭一餐 / 一天 / 外食  `(meal-structure)`
- 先修：nutrition-foundations, tracking-basics
- 目标（apply）：用「餐盘法」（半盘菜 + 一掌蛋白 + 一拳主食 + 一拇指脂肪）快速搭餐
- 目标（apply）：在外食 / 应酬场景做出赤字内的选择
- 练习（project）：用餐盘法设计你 3 天的菜单（含 1 次外食），发我；我核对是否落在你的赤字 + 蛋白目标内
- 里程碑：一份你愿意长期吃、达到赤字 + 蛋白目标的一周饮食框架 — 会了的标志：能不靠 app 精算，用份量法搭出落在目标内的一餐

## 阶段：competent

### 平台期与适应性产热怎么破  `(plateau-adaptation)`
- 先修：tracking-basics, nutrition-foundations
- 目标（analyze）：区分「真平台」（趋势体重 2-3 周不动）与正常波动
- 目标（evaluate）：用 diet break / 重新核算赤字 / 查依从性三步定位并破局
- 练习（drill）：假设你 3 周趋势体重没动，按「先查依从（是否真在赤字）→ 再核算 TDEE → 必要时 diet break」写出你的排查步骤，发我核对顺序

### 习惯与行为改变（不靠意志力）  `(habit-behavior)`
- 先修：meal-structure, movement-neat
- 目标（create）：用环境设计 + if-then 计划替代「意志力硬扛」
- 目标（apply）：把减脂动作拆成「微习惯」逐步叠加
- 练习（drill）：挑你最难坚持的一个减脂行为，写一条 if-then 计划 + 一个环境改造（如不买零食放家里），发我，我帮你压到「小到不可能失败」

### 依从性与情绪化进食  `(adherence-psychology)`
- 先修：habit-behavior
- 目标（evaluate）：破除「全或无」思维（破戒一餐 ≠ 整天报废）
- 目标（apply）：识别情绪化进食触发点并备一个替代方案
- 练习（drill）：写下你最常见的 2 个「破戒触发场景」，各配一个「if 触发 then 替代动作」，发我
- ⚠️ 诚实边界：如果出现暴食 / 催吐 / 极端节食 / 对食物和体重的强迫思虑 —— 这属于进食障碍的临床范畴，请停下并找精神科 / 临床营养师，不要靠这个课硬扛。

### 维持：从减脂过渡到长期不复胖  `(maintenance)`
- 先修：plateau-adaptation, adherence-psychology
- 目标（apply）：把热量逐步加回到维持（反向过渡），而非一停就暴食
- 目标（create）：建立长期监测红线（如趋势体重回升 X kg 就启动微调）
- 练习（project）：写一份你的维持方案：逐步加热量的节奏 + 一条体重红线 + 红线触发后的具体动作，发我核对
- 里程碑：独立设计并跑完一个可持续减脂 → 维持的完整闭环（赤字 + 蛋白 + 力量 + 追踪 + 平台调整 + 维持） — 会了的标志：能独立设计、执行、并在平台期 / 维持期自我调整一套不复胖的方案，并讲清每步背后的原理
