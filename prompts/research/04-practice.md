# Route 4 — 刻意练习 / 题库（含练习环境）

**Goal:** what the learner DOES per module — assessable exercises + the least-friction setup to practice.

**Find:** drills, problem sets, katas, project ideas for `{skill}`, graded by difficulty; and the
beginner practice environment.

**Where (high → low):**
- official exercises / the book's end-of-chapter problems
- established practice platforms (e.g. exercism, rustlings-style interactive courses)
- "build X" project lists from practitioners
- setup / "getting started" for the toolchain or materials

**Output → `references/research/04-practice.md`:** per module, ≥1 assessable exercise
(kind + prompt + what counts as pass); the env setup steps.

**Feeds:** module `exercises[]` — **every module needs ≥1** (pedagogy §3) — plus the curriculum's
setup module.
