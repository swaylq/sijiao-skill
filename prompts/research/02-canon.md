# Route 2 — 正典教材与课程

**Goal:** the canonical resources per concept, cross-validated.

**Find:** the books / courses / MOOCs / docs practitioners actually recommend to learn `{skill}` —
not SEO "top 10" lists.

**Where (high → low):**
- the resource itself (read its TOC / first lesson to confirm level + scope)
- named practitioners' "what I'd read to learn this" lists
- course platforms' real syllabi (not marketing copy)

**Rule:** a resource enters canon only with **≥3 independent** recommendations (see manifest).
Mark each `primary` (the resource itself) vs `secondary` (a review).

**Output → `references/research/02-canon.md`:** per concept, the 1-2 best resources with real
URLs, level (novice / …), and a one-line "why this one".

**Feeds:** module `resources[]`.
